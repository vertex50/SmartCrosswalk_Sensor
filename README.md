#[2016_한이음] 3팀_IoT기반 국도 횡단보도 보행자 지킴이

아두이노를 사용하는 센서 제어 및 제어장치와의 통신 알고리즘.

## 개발 환경
> 개발 언어 : c/c++ for Arduino 1.6.9

> 개발 타겟 : Arduino Mega 2560 R3

> 개발 도구 : Eclipse Mars.2, Github, Arduino
